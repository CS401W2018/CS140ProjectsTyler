CSS Basics

CSS - Cascading Style Sheets.

Inline CSS

Hello world! This is colored using inline CSS! You do it by doing <p style = "color: red;" >. Don't do this on my assignments!

Embedded CSS

Hello world! This is colored using embedded CSS! You do it by using the <style> tag! Than you can put in all your css there! I put the style tag within the header! Heres an example:

<style>
    .embedded{
        color: blue;
}
</style>
Don't use this either on my assignments!!!

External CSS

Hello World! This was colored using external CSS! To use this you want to add <link rel="stylesheet" href="index.css"> to your header! This is you saying "hey theres another file attatched to this one load it by going here!"
This is the superior way because it can help us save time! It helps you keep all of your web pages looking the same! It allows us to not have to copy and paste the same css, and allows us to use ONE file for all of our HTML pages! One change in this file changes all of the files! Also another reason is because of "seperation of concerns", a lot of computer scientists and programmers may not have the skills to make GOOD looking website, which is why we have graphic designers and user experience experts, and they if they want to go in and change how things look, can just go in and edit the CSS file without having to know HTML at all!
