1)

We need the viewport meta tag because it allows us to "tell" the web browser to stay the same size and to not like resize the webpage, we want a phone or something to use its screen dimensions and to not try to fake like being larger or something.

2)

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;" CSS will default to using the last verison of it, so if you have "flex-direction; column" first and "flex-direction: row;" 2nd, itll use the row.

3)

To change CSS for a web page displayed on a phone you can use "media only screen and (max-width: 800px)" at the bottom of your CSS. Within this we can put new styles for things like body, nav, the exact same way you add styles normally.
