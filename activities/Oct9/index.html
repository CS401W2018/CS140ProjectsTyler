  1. What is the difference between block tags and inline tags.

    The difference between block tags and inline tags are that block create a little "box" around the specific tag and that inline tags don't, they just kind of just function like text almost like this text I'm writing. So for example a <p> tag creates a little "box" around this text! but if i use a <strong> tag it makes that text bolded! It didn't create a box and seperate out that text from other stuff it just let it sit within the text.

  2. What is the margin, border, and padding?

    The margin is an invisible "ring" almost around a box created via a block tag! Its the space that seperates out text. So similar to how theres a giant "box" around all of the content on this page, i've made it so that the margin on the left and right "push" it towards the center. The border for a block tag is effectively the "outside" of the box. It is literally the border between that box and the next box. We can change the line thickness and such, and thats what i've done to this text! Thats what the box around it is, its the border. And then the padding is what allows us to "move" the text within the "box" of a block tag. Its what allows me to move the text within this to be slightly away from the edges of the border!
