  1. What is the difference between a block and an inline tag? Give examples of each one.

    The difference between a block and an inline tag is that block tags have "boxes" around them. Tags such as the div, p, body, main, header, etc. all are considered box tags, where they seperate itself via an invisible box around the content from other parts on the website. Inline tags are tags that function as text, similar to all this text within this paragraph, such as the strong, img, em tags. They don't seperate themselves from content because they are the content.
