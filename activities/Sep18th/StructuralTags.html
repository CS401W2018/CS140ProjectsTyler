Welcome to Web Development!

Header

The title and important information go in the header! The nav bar can also sometimes go in the header if you want. Basically any information that explains what the entire website is about!

Navigation

It should contain links to navigate around the website, should remain the same accross pages navigation stays the same between pages. The nav can go in the header or not, just depends on what the developer wants. But it also should only contain links to get around your website, generally we would never link a different website in the navigation bar.

Main Content

Main is where you put the content of your website so it should change between different pages on your website. This is the meat of your website, the stuff the user should interact with.

Footer

Footer - good for copyright, social media links, also cookie prefrences, to allow user to opt out of cookies also privacy notices and legal notices

HTML Validation

Firefox uses Mozilla Chrome and Safari use webkit. Easiest way to find a validator is to just search "HTML Validator" on google. We need to fix validation errors for projects but we don't need to worry about them for engagments!
