CSS Selectors

List out the different ways to select an item:

Tag name: Just the name of the tag such as a main tag. This will apply all the styling to EVERY one of those tags and EVERYTHING within that tag in the case of something like a main tag!

Descendent Rule: When you use something like "main p" where it selects all paragraph tags within our main tags

ID: < id="IDName" > you can use ID's to name any specific tag! Works in header tags! Works in paragraph tags! Works in everything! To edit via CSS use "#IDName".

Class: Works almost the exact same as an ID but you use a "." to edit it in the CSS, Example: ".ClassName{css stuff here}". You can add multiple classes as well to a tag using a space between them. Example: <p class="first second">, this adds the class of "first" and the class of "second" to that paragraph tag!

What is the difference between an ID and a Class:

ID: To edit via CSS use "#IDName". YOU CANNOT HAVE SPACES IN ID's. Use a "-" or "_" or a version of camel case. ID Should only be used ONCE. Don't have multiple tags using the same ID! Usually uses a noun as a name.

Class:Shares some rules as with ID's, to not use spaces in the naming!!! Can be used as many times as you want! Use this if theres multiple tags that have the same class! So this is reusable but ID's aren't reusable. Usually uses an adjective as a name.

What is the difference between a SPAN tag and a DIV tag:

You can use the <span> tag to help edit certain text! A span tag doesn't do anything by itself but it allows you to attatch an ID or a Class to it, allowing you to style certain parts of text! Use a span tag when you don't have a lot of tags within it! Useful when styling specific parts of text.

For Div tags you want to use these when you have lots of tags within it! Div tags DO cause line breaks!!! Makes a little box almost of the text within it thats its own thing. Heres an example of how to use one: <div class="classname" id="IDName">

What is the difference between a DIV tag and a HEADER, MAIN, FOOTER:

They are all effectively the same. The main difference is that the header main and footer tags allow things like google to not have to look at your class or ID names to be able to figure out whats going on in your webpage! In the past people use to just use only Div tags and add in an id or class name of "Main" or "Header" or "footer" and such.

What are different ways to select a color:

You can use RGB values, can also just use names of the color like "white" or "black" or "red" and such. Didn't get to it though in class.*
