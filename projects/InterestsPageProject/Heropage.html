My Hero - Brackeys

Brackeys is one of the most prominant youtube channels in the game development scene. While he doesn't post often if at all anymore, his channel provides some of the best tutorials there are on how to learn and do certain things in game development. He has been an insane help to me as I've grown my game development and coding skills, and I know he has been a major help to many others as well when they are learning and growing their own game development skills.
Heres a list of some of his most popular videos:

Video 1 - How to make a Video Game - Getting Started (Unity)
Explains what game engines are and how to pick which one you want to use when making your first project
Video 2 - How to make a Video Game in Unity
He goes over how to download and install unity, as well as getting you setup with your first project!
Video 3 - How to make a 2D Game in Unity
Helps get you started with 2D game creation and sets you up with all the assets you will need as well for a basic project
